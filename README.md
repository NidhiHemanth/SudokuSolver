# Sudoku Solver

Automated sudoku solver using BackTracking algorithm.\
GUI in the form of a game made using PyGame.

## Requirements

Python module : pygame

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pygame.

```bash
pip install pygame
```
